# VOID 🌑✨
### Modern Minimalist Developer Portfolio

"Void" is a high-performance, strictly monochrome developer portfolio designed for maximum visual impact and professional elegance. Built with **React**, **Vite**, and **Tailwind CSS**, it features a distinctive "Obsidian Night" aesthetic combined with smooth Lottie animations and a robust Supabase backend.

---

## 🌑 Aesthetic Concept
The portfolio follows the **"Void Aesthetic"**—a deep obsidian monochrome theme (`#030303`) that eliminates visual noise and focuses entirely on content and typography. It utilizes:
- **Obsidian Black & Clean White** palette.
- **Glassmorphism** UI components.
- **Micro-interactions** and smooth AOS transitions.
- **Cinematic Vector Animations** via DotLottie.

---

## 🚀 Technical Stack
- **Frontend:** React.js + Vite
- **Styling:** Tailwind CSS + Lucide Icons
- **Backend:** Supabase (PostgreSQL)
- **Animations:** DotLottie, AOS (Animate On Scroll), Framer Motion
- **SEO:** React Helmet Async
- **Audio:** Custom SFX Transition System

---

## ✨ Key Features
- **Project Showcase:** Dynamic project cards synced with Supabase.
- **Interactive Details:** Rich project detail pages with features and tech-stack arrays.
- **Real-time Comments:** Live commentary system with image upload and admin pinning.
- **Welcome Experience:** Immersive welcome screen with synchronized cinematic SFX.
- **SEO Optimized:** Optimized meta tags and semantic HTML for search engines.
- **Fully Responsive:** Seamless experience across mobile, tablet, and desktop.

---

## 🛠️ Setup & Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/kyywxv/void.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

---

## 📤 Deployment
This project is optimized for deployment on **Vercel** or **Netlify**. Ensure that you add your Environment Variables (`VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`) in the platform's deployment settings.
